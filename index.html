Operations Research: Chapter 1

Cheat Sheet & Summary | ملخص للمراجعة السريعة

1. Definition & Origins (التعريف والنشأة)

Concept Definition (English) الشرح (بالعربي)
Operations Research (OR) The scientific approach to problem-solving and decision-making for executive management. النهج العلمي لحل المشكلات واتخاذ القرارات للإدارة التنفيذية.
Goal of OR To find an Optimal Solution (best possible solution) under given constraints. الهدف هو إيجاد "الحل الأمثل" في ظل القيود الموجودة.
Origin Started during World War II (Military operations), then moved to industry (post-1950s). بدأت خلال الحرب العالمية الثانية (تطبيقات عسكرية)، ثم انتقلت للصناعة.
Management Science Another name for Operations Research. اسم آخر لعلم بحوث العمليات (علم الإدارة).

2. The OR Process Methodology (منهجية الحل)

* Important: Memorize the order of these steps (هام: احفظ ترتيب الخطوات)

1
Defining the Problem: Identifying the objectives and constraints clearly. تعريف المشكلة: تحديد الأهداف والقيود بوضوح.
2
Developing a Model: Translating the problem into a mathematical representation. بناء النموذج: تحويل المشكلة إلى تمثيل رياضي.
3
Acquiring Input Data: Collecting accurate data for the variables. جمع البيانات: الحصول على بيانات دقيقة للمتغيرات.
4
Solving the Model: Using algorithms/software to find the solution. حل النموذج: استخدام الخوارزميات لإيجاد الحل.
5
Testing the Solution: Checking if the solution is realistic and valid. اختبار الحل: التأكد من واقعية وصحة الحل.
6
Analyzing the Results: "What-if" analysis (Sensitivity Analysis). تحليل النتائج: تحليل الحساسية وتأثير تغيير المتغيرات.
7
Implementation: Applying the solution in the real world. التطبيق: تنفيذ الحل على أرض الواقع.

3. Classification of Models (تصنيف النماذج)

Model Type Description Example
Iconic Model
نموذج أيقوني		 Physical replica / Scale representation. Toy car, Photograph, Globe.
مجسم سيارة، كرة أرضية
Analog Model
نموذج تناظري		 Physical, but measures abstract quantity. Thermometer, Speedometer, Clock.
ميزان حرارة، عداد سرعة
Mathematical Model
نموذج رياضي		 Uses symbols and equations. Profit = 10x - 500
المعادلات والرموز

4. Data Categories (أنواع البيانات)

Deterministic Models (نماذج حتمية)

All input data is known with certainty.

Example: Raw material cost is fixed.

جميع البيانات معروفة مسبقاً وبشكل مؤكد.
Probabilistic Models (نماذج احتمالية)

Input data is uncertain or involves risk/probability.

Example: Future customer demand.

البيانات غير مؤكدة وتعتمد على الاحتمالات.

6. Quantitative Analysis Terms (مصطلحات هامة)

Decision Variables
متغيرات القرار		 Controllable variables (e.g., How many units to produce).
الأشياء التي نتحكم فيها (كم ننتج؟).
Objective Function
دالة الهدف		 The goal equation (e.g., Maximize Profit or Minimize Cost).
معادلة الهدف (تعظيم الربح أو تقليل التكلفة).
Constraints
القيود		 Limitations (e.g., Budget, Labor hours, Raw materials).
المحدوديات (الميزانية، ساعات العمل).
Feasible Region
منطقة الحل الممكن		 The set of all possible solutions that satisfy constraints.
جميع الحلول التي تحترم القيود.
Optimal Solution
الحل الأمثل		 The single best point in the feasible region.
أفضل نقطة تحقق الهدف.
